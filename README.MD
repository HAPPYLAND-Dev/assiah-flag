# Assiah Flag

## 描述
HAPPYLAND 服务端通用启动参数

## 特性
- 支持自选目录下的多个服务端
- 使用 `ZGC` 取代 `G1GC`
- 神秘特调启动参数
- 自动重启
- 方便修改内存

## 安装
直接取代你原本的启动脚本然后 `./Start.sh` 选择服务端文件并回车
